computer is an electronic data processing machine.

Computer science focuses on the theory of computation, using algorithms, data structures, and programming languages to create software, manage hardware, and build networks.

Basic Components

  1. Hardware

  2. Software

Hardware Components

  1. Central Processing Unit(CPU)

    • Computer's brain; processes data/instructions.

  2. Input Unit

    • An input unit accepts data.

  3. Output Unit

    • The product of a process.

Software Components

  1. System Software

    • Controls the computer's hardware and resources.

  2. Application Software

    • End-user programs performing specific tasks.

  3. Programming Software

    • Develops applications and systems.
